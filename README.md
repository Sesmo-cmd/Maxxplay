# Maxxplay
A browser game built entirely with HTML5 Canvas and vanilla JavaScript — no frameworks, no libraries. Features 3 playable zones, unique boss battles, 6 enemy types, chiptune sound engine, smooth physics, and mobile touch controls. Inspired by Super Mario Bros and GTA VII
# 🎮 Super Mario: Vice City

> It's Mario. But he's in Vice City. Stomp thugs, dodge cops, collect cash and fight bosses across 3 neon-drenched zones. One HTML file. No install. Just play.

## ⚡ Features

- **3 Playable Zones** — each with a unique aesthetic, enemies and boss fight
- **6 Enemy Types** — Goomba, Koopa, Thug, Sniper, Biker, Tank
- **Boss Battles** — HP bars, phase 2 rage mode, multi-bullet attacks
- **Wanted System** — 5-star wanted level, cops spawn and hunt you down
- **Cash Economy** — stomp enemies and collect cash bags to grow your score
- **Power-Ups** — Gun mushroom and Star power
- **Chiptune Audio** — custom sound engine with zone music and SFX
- **Mobile Controls** — on-screen D-pad and action buttons
- **Smooth Physics** — coyote time, variable jump height, camera easing
- **Minimap** — real-time world minimap with enemy positions

---

## 🕹️ Controls

### Desktop
| Key | Action |
|-----|--------|
| `Arrow Keys` / `WASD` | Move |
| `↑` / `W` | Jump (hold for higher) |
| `Space` | Fire (when armed) |

### Mobile
On-screen D-pad and buttons appear automatically on touch devices.

---

## 🚀 How to Play

1. Click **Play Now** above — or download the HTML file and open it in any browser
2. Stomp enemies for cash
3. Grab the **Gun Mushroom** to unlock shooting
4. Grab the **Star** for invincibility
5. Reach the end of each zone and defeat the boss
6. Survive all 3 zones to win

---

## 🛠️ Built With

- **HTML5 Canvas** — all rendering, no WebGL
- **Vanilla JavaScript** — zero dependencies, zero frameworks
- **Web Audio API** — procedural chiptune sound engine
- **CSS** — UI, HUD, mobile layout

> Single file. No build step. No npm. Just open and play.
